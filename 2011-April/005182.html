Charles Curley <charlescurley at charlescurley.com>:
> If I understand the man page for scons, the way to turn off
> compilation of a driver is, e.g.:
> 
> scons tripmate=false
> 
> However, when I build with that, the tripmate driver is still in the
> build.
> 
> $ ./gpsd -l | grep -i tripmate
> 			Delorme TripMate
> 
> Am I doing something wrong or is that not yet working?

scons --disable-tripmate

Your confusion is reasonable. tripmate=false would be the correct form
if we were using the older native scons convention for arguments. I
chose instead to make the CLI more configure-like, using a newer
argument-declaration feature that's based on Python optparse.
