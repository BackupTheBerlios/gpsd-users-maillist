Host PC: PIII 1GHz with Gentoo 2007.0
GPS: Garmin 18LVC running in NMEA mode (had trouble with early revisions 
when running garmin serial)

Test procedure:

Checkout desired SVN revision into clean directory
cd into revision directory
run autogen.sh & make
start the freshly compiled gpsd using
	./gpsd -nN -D3 /dev/ttyS1
change consoles
start telnet
send 'do' command until gpsd has locked and is giving sane data
send 'do' command and exit telnet as quickly as possible. Take note of 
timestamp (1)
wait approx 2 minutes
start telnet
send 'do' command
note timestamp (2)
immediately send 'do' again
note timestamp (3)
exit telnet

If the error is present, timestamp (1) & timestamp (2) are almost identical 
(depends on how quickly you exited telnet) and Timestamp (3) should be 
approx 2 minutes (or the duration between telnet exit & restart) later than 
timestamp (1) & (2).
