Generally a GPS is not on the hardware list because we haven't got any
test reports. If it works, we'll put it up. If it doesn't work, we'll
note that too.

We try to avoid being too distribution-specific, but we will take
advantage of useful features where we can. By way of example, the
garmin_usb driver is apparently only available on linux, and the tty
timestamping feature is only available under openbsd, yet we still
have ways to have them in the generic code base.

We sincerely hope that nothing ubuntu-specific needs to be done to
make gpsd work... you should be able to point it at a tty and
rock-n-roll. If you're feeling brave, you can check out gpsd from
SVN... a number of useful things have gone in since 2.34.
