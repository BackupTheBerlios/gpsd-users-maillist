sub initialize {
  my $self = shift();
  my $data = shift();
  foreach (keys %$data) {
-    $self->{$_}=$data->{$_};
+    $self->{$_}=[@{$data->{$_}}];
  }
}

For you non-perl type [@array] makes an anonymous array reference
(makes a copy) vice \@array which is a hard array reference.  $data is
a reference to a hash of array references.

e.g.
{S=>[0]
 P=>[38, -77]}
