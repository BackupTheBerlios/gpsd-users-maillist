vwf <vwf at vulkor.net>:
> I get data from a pilot plug. As far as I can see perfectly good data,
> but gpsd does not seem to like it: xgps does not show position or AIS.
> Can someone tell me what can be wrong?

I used gpsfake to repeatedly loop through the data, running xgps.

The reason you're not seeing any AIS data in the window is because 
the log only contains type 1s.  Presently xgps assumes that the 
data stream will include Type 5s and holds off displaying type 1s 
until it has a ship name for the corresponding MMSI.

If Pilot Plugs do not pass through type 5s perhaps I need to change
xgps so it doesn't assume it will see them.
