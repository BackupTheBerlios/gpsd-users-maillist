I am no dbus expert, but that never stopped me from having an
opinion. Some thoughts:

gpsd does have a dbus interface, but you can compile without it:

./configure --disable-dbus

I'd try that and see if the problem goes away. Of course, if the
problem is intermittent, that might not help.

An alternative would be to turn on logging and see what shows up. You
might look at how the dbus interface is instrumented first.

Is there a general purpose dbus monitor program that will dump all the
traffic?
