that's a python-ism.

The module gps has defines an object called gps.

If you import the module you have to prefix the class with the module name:

$ python
Python 2.6.5 (r265:79063, Apr 16 2010, 13:09:56)
[GCC 4.4.3] on linux2
Type "help", "copyright", "credits" or "license" for more information.
>>> import gps
>>> session = gps.gps()
>>>

as a short-cut you can import the names from the module directly:

$ python
Python 2.6.5 (r265:79063, Apr 16 2010, 13:09:56)
[GCC 4.4.3] on linux2
Type "help", "copyright", "credits" or "license" for more information.
>>> from gps import *
>>> session = gps()
>>>

Based on your email I thought you'd done the second and the "gps."
prefix wasn't needed.
