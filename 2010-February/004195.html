> Grepping the code for fork() and pthread_create() show me no other

I don't think that it hangs, because if I start with -D 2, the gpsd
process with the lowest PID is the one that writes to the syslog.

FYI, this is the part of my /etc/init.d/S50gpsd file that starts gpsd:

---8<---

NAME=gpsd
DAEMON=/usr/sbin/$NAME
DEVICES=/dev/ttyS3
PIDFILE=/var/run/$NAME.pid

start() {
        echo -n "Starting $NAME: "
        start-stop-daemon -S -q -p $PIDFILE --exec $DAEMON -- -n -D 2
-P $PIDFILE $DEVICES && echo "OK" || echo "Failed"
}

---8<---
