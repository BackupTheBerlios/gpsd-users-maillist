Just a quick  follow up for others using Ubuntu/Kubuntu  7.10.    There 
is indeed (as Gary suggested) a precompiled module for Garmin gps which 
works with my Garmin GPS18 usb puck.   Apparently the module comes with 
the default 7.10 gutsy installation.  I didn't have to compile it.   It 
is located at 
/lib/modules/2.6.22-14-generic/kernel/drivers/usb/serial/garmin_gps.ko
Load the driver aka kernel module with this command (entered in a 
terminal window)
$ sudo modprobe garmin_gps.ko

Then when you attempt to test your gps with
$ sudo gpsd -nND4 /dev/ttyUSB0
you'll be rewarded line after line of data from your gps device.

xcgs and cgs also now work.  These programs are part of the Ubuntu 
package "gpsd-clients" which can be installed from the repository 
"universe/misc" with Adept Manager.  After installation and modprobing 
gpsd,  just enter xgps or cgps in a terminal or Konsole window. 
(KdeStart -> System -> Konsole Terminal Program). ( BTW  Settings 
->Schema->Black on LightYellow gets rid of the awful white on black User 
Interface which is the default for the Konsole window).
