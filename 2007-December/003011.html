Larry on the Dell Laptop <larry at foxgulch.com>:
> Just a quick  follow up for others using Ubuntu/Kubuntu  7.10.    There 
> is indeed (as Gary suggested) a precompiled module for Garmin gps which 
> works with my Garmin GPS18 usb puck.   Apparently the module comes with 
> the default 7.10 gutsy installation.  I didn't have to compile it.   It 
> is located at 
> /lib/modules/2.6.22-14-generic/kernel/drivers/usb/serial/garmin_gps.ko
> Load the driver aka kernel module with this command (entered in a 
> terminal window)
> $ sudo modprobe garmin_gps.ko
