On Sat, 22 Apr 2006, Christophe Vandeplas wrote:

> A maybe interesting thing is that I checked the I/O format in the GPS and
> it's set to GARMIN format,

Garmin format only works with the Garmin/USB kernel driver.  I did not seat
in your earlier debug.

Try this:

# cat /proc/bus/usb/devices


T:  Bus=02 Lev=01 Prnt=01 Port=00 Cnt=01 Dev#=  2 Spd=12  MxCh= 0
D:  Ver= 1.10 Cls=ff(vend.) Sub=ff Prot=ff MxPS= 8 #Cfgs=  1
P:  Vendor=091e ProdID=0003 Rev= 0.01
C:* #Ifs= 1 Cfg#= 1 Atr=80 MxPwr= 76mA
I:  If#= 0 Alt= 0 #EPs= 3 Cls=ff(vend.) Sub=ff Prot=ff Driver=garmin_gps
E:  Ad=81(I) Atr=03(Int.) MxPS=  64 Ivl=1ms
E:  Ad=02(O) Atr=02(Bulk) MxPS=  64 Ivl=0ms
E:  Ad=83(I) Atr=02(Bulk) MxPS=   8 Ivl=0ms

Notice the Driver=garmin_gps.  If that is not there then you MUST use NMEA.

Your best bet is NMEA.
