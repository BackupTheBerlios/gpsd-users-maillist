We compile on a chrooted environment with compiler that come on the cd-rom
of Technologic System Inc for development for the TS-5500.

We use gcc 2.95.3 in this environment.

We have kernel version 2.4.34 patched for TS-5500 board of Technologic
System Inc.

We upgrade the kernel 2.4.23 that come from Technologic System Inc to have
new driver for USB to serial converter PL-2303 for the GlobalSat Bu-353 gps
and also for the driver USB to serial FTDI serial convertor for the USB
Crystall Fontz 4x20 external LCD model 634 XE.

We know that actually the last version of kernel is 2.4.37 that have new
version of PL2303 driver, but we don't try to port this kernel to the
TS-5500.
