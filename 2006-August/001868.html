Hi,

You need to parse VDM message number 5. It is what you are looking for.
A VDM message is often splitted into several sentences.

They might look like this:
!AIVDM,2,1,2,A,53u1V`01gnR5<DTn221>qB0thtJ222222222220l0pJ644b?e=kSlTRk,0*0E
!AIVDM,2,2,2,A,l2CQp8888888880,2*22

Notice that the first character in the message is "5". It means that
this is a "Ship static and voyage related data".

I have written a parser for VDM messages 1,2,3 (position report "dynamic
data") and 5 (name/IMO-number/draught/destination etc "static data") in C.

I will make it public available (under an Open Source license) when I
consider it is ready.

ITU Recommendation M.1371
(http://www.itu.int/rec/R-REC-M.1371-2-200603-I/en) explains how the
messages are encoded.

Read "3.3.8.2.3  Message 5: Ship static and voyage related data" for
more information.

You might find IALA TECHNICAL CLARIFICATIONS ON RECOMMENDATION ITU-R
M.1371-1
(http://www.navcen.uscg.gov/enav/ais/ITU-R_M1371-1_IALA_Tech_Note1.3.pdf)
useful too.
