The libgps man page, dated 14 Aug 2004, shows the prototype for
gps_set_raw_hook (really, for the hook function) incorrectly.

The man page says:

   void gps_set_raw_hook(struct gps_data_t *gpsdata,
             void (*hook)(struct gps_data_t *, char *buf));

However, gps.h ($Id: gps.h 5488 2009-03-17 20:54:34Z esr $) says:

void gps_set_raw_hook(struct gps_data_t *gpsdata,
     void (*hook)(struct gps_data_t *sentence, char *buf, size_t len, int level));

This is on gpsd	2.39-5 on Ubuntu.

Also, it would be nice to know what the various parameters to the hook
function are good for.
