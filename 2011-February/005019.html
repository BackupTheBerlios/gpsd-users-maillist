I haven't got my hands on any GSD4e devices so I don't really have
anything useful to add to the discussion.

Please have a look at http://gpsd.berlios.de/faq.html#willitwork
Perhaps the only thing I'd add to that section is to check the
protocol reference for the new chip - it seems safe to assume that
unless the protocol reference says that there are significant changes
to support new hardware, new revisions of a chip won't break the. By
way of example, there were very few important changes between uBlox'
Antaris and Antaris4 or SiRF's SS2 and SS3.

So what I'm suggesting here is that you talk with your Vendor, OEM,
etc. to get the protocol reference and see what (if anything) changed.
