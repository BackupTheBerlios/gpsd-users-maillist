Calum,
A small script like this (I didn't test this code on a machine) should
work for you.  You could even record your own voice and play "you've
got fix".
Mike

perl -e '
 use DateTime::Precise;
 use Net::GPSD;
 $gps=new Net::GPSD;
 my $last;
 while ($gps->get) {
    if ($_->fix != $last) {
      print "\007";
    }
    $last=$_->fix;
    sleep 0.5;
 }
'
