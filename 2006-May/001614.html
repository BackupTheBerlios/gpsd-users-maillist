Try typing: "killall -v gpsd"

If that doesn't return anything, then try: "sudo killall -v gpsd"

Your old copy of gpsd is probably still running in memory, although 
you've replaced it on disk.

Also, if you like using gpsd on a Mac, check out 
http://www.grandhighwizard.net for some GUI utils (and a Universal 
Binary of gpsd).

Cheers,
Robin
