Since GPSD 2.30 is not compatible with Kismet I've compiled 2.10 which 
is the last version before the baud-matching logic was introduced and 
seems to be the most stable with Kismet. It's compiled for NMEA protocol 
and should install without issues as it's a single binary, but does 
carry dependencies as listed by the ipkg.

Check it out here.  
http://www.frontiernet.net/~beakmyn/openwrt/package/gpsd/gpsd_2.10_mipsel.ipk

Below contains the source ready to go
