http://cnds.ucd.ie/~cetus/magellan/comms.html

$PMGNST,mm.nn,t,f,bbb,hh.h,oooooo,pp*cc<CR><LF>

Field	ID	Description
1	mm.nn	Software version number
2	t	Type of last fix (0- none, 1-init, 2 -2D, 3-3D)
3	f	T if first fix, F if no fix yet
4	bbb	Battery voltage level (0-999) in arbitrary units
5	hh.h	Hours of operation left on battery (approximate)
5	oooooo	Oscillator offset in Hz at L-band
7	pp	PRNprietary information
