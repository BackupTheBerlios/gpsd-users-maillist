The Rayming site as a Sirf utility program that you run on a PC and it
allows the unit to be reconfigured. It also appears to allow the firmware
in the unit to be updated. Not sure where you would get firmware updates
from.

I played around with this but is appears that any changes you make will be
lost when you power cycle the device.

This means the the driver must always setup the unit to use WAAS for
correction and tweak any of the other settings you might want to change.

The Rayming site also has the SiRF data sheet which lists all the
registers and outlines the binary com mode. It does not have a good
description of what the various settings in the registers actually mean or
what it will do.

The utility is very frustrating to use and takes many attempts in my
experience to sync up with the GPS unit before it starts to work
correctly.
