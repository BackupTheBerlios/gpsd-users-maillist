Yo Glen!

Looks like your C compiler is not properly set up to find all your
include files.  Not sure how this works on the Mac, but with Linux
you would need to add /usr/X11R6/include to the CPATH variable
in your environment.

With Linux, if you use bash, you would set this in your /etc/profile:

export CPATH=$CPATH:/usr/X11R6/include
