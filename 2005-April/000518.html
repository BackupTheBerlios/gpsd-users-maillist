On Thu, 31 Mar 2005, Eric S. Raymond wrote:

> So you were just being condservative/paranoid.  That's not a bad thing.

You rememnber when the first time Sun shipped Solaris on x86?  That got
canceled a long time ago, now it is back and not seen the new code.

Anyway, a client of mine had a LOT of Intel boxes running that code.
His VC were involved in Intel and Sun so he had no choice.  Well the
Solaris on Intel code always did strange things.  Icons flipped weird,
cursors looked like venetian blinds, little used code sections would
crash.  They would get it fixed and then new patches would have the
same bug.  Obviously the porting team had no leverage on the patching
team.  It was so bad the Sun just canceled it and buried the mistake.

Since then I never assume code is portable until it has been shaken
really hard and stirred.
