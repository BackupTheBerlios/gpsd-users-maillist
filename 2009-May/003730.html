Hi,

Seth Haines wrote:

> When I kick off gpsd with the options:
> [root at mobile3 ~]# gpsd -D9 -N -F /dev/ttyUSB0

   That command line tells gpsd to create a control socket (i.e., accept
user commands) on /dev/ttyUSB0.

   What you probably want instead is this:

gpsd -D9 -N /dev/ttyUSB0
