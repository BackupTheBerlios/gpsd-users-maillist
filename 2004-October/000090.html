Hi Ramiro,

I suggest gpsbabel at http://gpsbabel.sourceforge.net/  It runs under 
Windows, Linux, OS X, and misc other unix like OS's.

If you set the Garmin output to 'garmin' format, this line will fetch 
tracks from a Windows machine with the GPS on the first serial port:

gpsbabel -t -i garmin -f com1 -o gpx -F yourfile.gpx

Change com1 to the device name for a *ix sort of machine.

On my ibook:
-f /dev/cu.usbserial0

on linux 
-f /dev/ttyS0


Cheers,
Rich
