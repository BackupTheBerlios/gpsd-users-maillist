Greetings,

   this is to announce that qtGpsc, a gpsd client I wrote some time ago
has finally been rewritten to: a) use the latest Qt4, rather than the
then already obsolescent Qt3, and b) communicate with post 2.90 gpsd.

   In fact, this new client uses libgps API version 5.x so it requires
libgps20, currently only available from trunk. libgps19 and older are
not supported, in the interests of simplicity and due to time
constraints (however anybody who wants to is of course free to code in
support).

   The source code and documentation are available at:
http://code.google.com/p/qtgpsc/

   Note that this is in all likelihood the last qtgpsc version not to
have any KDE dependencies.  I plan to fully integrate this client with
the KDE desktop in future versions.


Regards,
