> Only the SL6000 has built-in wifi. All of the others can do it via a

> I would suggest the C3x00 if you want lots of storage space (C3200 if

WiFi seriously burns the battery. Much worse than a USB gps. I've got
about 3 useful hours running a datalogger on my SL-C3000 while it was
powering a BU353 (SiRF3 usb gps). That's with the backlight on, the
processor at full speed and the disk spinning. I suspect that if I had
been using CF for my disk, had dimmed the display, buffered more data
in memory and slowed the CPU, I could have extended that to about
6hrs.

One guy I know has an external battery pack... has 4 1.25V 22mAH NiMH
cells in it and that gives him huge amounts of runtime without having
to resort to insane powersaving measures.
