> *BSD-Guys: How is this solved under OpenBSD/NetBSD? AFAIK the path

AFAIK there is no Garmin USB driver for OpenBSD or NetBSD.  W/o a
kernel driver for the Garmin USB there is no way for gpsd to connect to
a garmin USB.  So for now the Garmin USB support in gpsd is Linux only.
