Yo Mark!

On Mon, 16 Jan 2006, mark_2811 wrote:

> I get the error "can't get "tcp" protocol entry" when I try to launch it.

You are missing this line from your /etc/protocols file:

tcp     6       TCP     # transmission control protocol

Maybe you are missing the entire file?


And can you verify that your "gpsd" is a binary file and not a script like
you had before?

This will find out which gpsd is running:
# which gpsd

This will verify it is a binary:

# file `which gpsd`

Here is what mine shows:

/usr/sbin/gpsd: ELF 32-bit LSB executable, Intel 80386, version 1 (SYSV), for GNU/Linux 2.0.0, dynamically linked (uses shared libs), not stripped
