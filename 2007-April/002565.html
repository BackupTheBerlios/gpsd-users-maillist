All,

I've noticed a couple of errors in the libgps.xml/libgps.3 man page.
The SYNOPSIS section lists the correct name of the functions (such as
gps_open()), but the DESCRIPTION section uses names with a leading
"gpsd_" instead of "gps_" (such as gpsd_open()).

It appears that the XML files are the master copies which are used to
generate the man pages, so I have a provided a patch against the
libgps.xml shipped with 2.34.

The libgpsd.xml/libgpsd.3 man page appears to be OK.

Matt Roberds
