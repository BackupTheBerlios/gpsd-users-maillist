Yes, that was the only *compiler related* problem I encountered - all 
other problems came from library stuff.

Of course this can easily be solved by using something like:

FIELD_INIT(.field =) value

and a corresponding definition like:

#ifdef _MSC_VER
#define FIELD_INIT(init)
#else
#define FIELD_INIT(init) init
#endif

or a compiler specific define in gpsd_config.h - whatever seems more 
appropriate.


So that's really a solvable problem.

But the question for me in the end is: Does such solutions fall under 
the "and doesn't mess up the rest of the source tree" that Chris mentioned?
