I have one of these. The only way I have found to get this back the NMEA 
mode is let the battery run down and sit for a while. Some say you can 
change it with gpsd but this does not work. Somehow gpsd changes the 
mode to some other mode than NMEA and then gpsd can't read the output. I 
can't prove that gpsd is causing the problem. It could be bluez. My 240 
worked great for a while but now no.

You don't have to run minicom you can 'cat /dev/rfcomm0'. If you get 
output then you are connected. You will be able to detect NMEA mode 
easily. There are readable lines. Other mode/s will be junk.

I know this does not help much but I have other holux gps devices that 
work. It may be a bug in the 240.

Well, good luck,
