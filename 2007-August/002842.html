    double online;               -----> what is used for and how to use it?

This is basically a time stamp of the last time data was received from the gps.  I use this information to make sure the gps is connected as this value will continually change if the gps is working.

NOTE:  If you use gps_query(), this value will change every time whether the gps is connected or not.   In order to fix this, I followed Mat Roberds advice

change line
526 of libgps.c (as shipped with gpsd-2.34) from

     received = gpsdata->online = timestamp();

to

     received = timestamp();

and recompile

This gives the behavior I expected.  Speaking of which, will this change be incorporated into 2.35?
