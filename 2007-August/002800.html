Philip Kopp wrote:

> The GPS is a Garmin GPSMAP 60Cx which has both USB and a serial
> interface. I also tested with a Garmin GPS V and a GPS III+ which have
> only serial interfaces.

I am using gpsd v2.34 with a GPSMAP 60CSx (not sure if it similar to the 
60Cx).  Things work well for me.  Make sure you have the garmin_gps 
module loaded.  To verify as root:

	lsmod | grep garmin_gps

You should see the module listed.

The SVN version of gpsd does not yet work with this gps.  This was the 
case a few weeks ago, anyway.
