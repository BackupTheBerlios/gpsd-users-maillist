Priya,

When I cross compiled for a powerpc I just had to use the "host"
parameter. I suppose you would know better than I would whether your
tools require you to specify target as well. An easy thing to miss is
making sure that your PATH and any other relevant environment variables
allow configure to find your build tools (such as pointing to the bin
directory where gcc and other tools are). Sometimes the libraries need
to be specified correctly too such as with LDFLAGS variable. Sometimes
it can try to use your build system's libs instead of what you intended.
