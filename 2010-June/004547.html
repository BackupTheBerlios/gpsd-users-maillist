Hi!

I use a Jentro BT-GPS-8 bluetooth GPS receiver. It works only when
using the -b switch otherwise it crashes refusing any further
connection.

Now I discovered it has a build in fixed port speed so when compiling
gpsd with ./configure --enable-fixed-port-speed=57600 gpsd even manages
to work and reconfigure the GPS receiver without -b switch.

Consequently my feature request:
Please add an baud rate option to the -b switch or to gpsd generally -
so it will work without recompiling by simply providing eg. -b 57600 -
I think this may circumvent problems with some bluetooth GPSs and makes
gpsd more user friendly as you don't have to recompile it as an
ordinary user.


Thank you and
regards,
  Andreas Weller
