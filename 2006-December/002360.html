Hi Carl,

Just be aware that if you want Pulse Per Second (PPS) output for use with a 
reference time service, you HAVE to buy the "bare-wire" (LVC) variant, as 
the PPS signal is NOT brought out on either the USB or PC variants.

The LVC variant has 'TTL level' RS232, and requires a 5V power supply and 
yourself to put the appropriate connector on the end if the cable. The LVC 
speaks NMEA natively, and is supposed to also speak the Garmin binary 
protocol. Did briefly try using binary a few months ago to test some of 
Gary's code but didn't have much success before the real world intervened 
and I went back to NMEA.

I've used a few of these - actually have one providing time data in 
Antarctica at the moment. Last report was that it was running OK.

ciao, Dave
