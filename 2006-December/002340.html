Ali Utku Selen <selenau at kentkart.com.tr>:
> I checked the documentation for lea-4s and it says module is working on NMEA
> and has a proprietary message for switching to ubx. So how will gpsd behave
> about this? Am I going to start gpsd with an option?

No; gpsd is totally autoconfiguring.  Until we teach it the new uBlox
protocol, it will read the device in NMEA mode. After we teach it, it
will recognize and use the uBlox binary protocol without any prompting
from you.
 
> I'm using xgps to test the device, what should I do in order to set "j=1" ?
> Use gpsctl? or can modify the xgps?

xgps -j should do it.  gpsctl will enable you to change modes once we get the
uBlox driver in place, but you'll never need to do that to make gpsd work.
