This wouldn't be the first time we've had the discussion about how
many numbers do we propagate. I'm in favor of passing everything -
because maybe, just maybe, today we are using an RTK or dual-frequency
receiver.

And any serious analysis program is going to check the quality flags anyway.

About the only thing I think we really need to check is that we handle
the mode properly. 4 = PPS, 6 = DR, ... is there anything we need to
watch for in the RTK case, or can we just treat it as normal DGPS?
