On Thu, 21 Dec 2006, Carl Brown wrote:

> I'm thinking about buying one of these:
> http://www.garmin.com/products/gps18oem/
>
> It is available in a db9 serial version which will output NMEA0183 or Ga
> rmin,

I have an 18/LVC and an 18/USB.  Both can work with gpsd in Garmin
Binary mode.

The advantage of the 18/LVC is that it does PPS and the disadvantage is
the need for 5V external power.

The advantage of the 18/USB is that it is powered off the USB and so is
very portable, the disadvantage is no NMEA or PPS.

> Does anyone have any opinions about NMEA vs Garmin in regards to working
>  with
> gpsd?

They both work fine with gpsd.  A lot of other projects only understand
NMEA so that is a problem fo the 18/USB.  If you want PPS time you have to
use the 18/LVC.
