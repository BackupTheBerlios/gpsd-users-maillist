Gary,

This is not an out-of-the box GPS.  It is the Venus634LPx chip on a
development board from Sparkfun (sku: GPS-09133).  The original test circuit
that I drew up looks for DTR to be asserted before it sends 3.3v to the
Venus board's VCC pin to power it on.  The Venus board also has a VBAT pin
for constant power which if kept hot will allow ~1 second startup time on
the GPS receiver.

I also have a serial GPS receiver in a Toughbook CF-19 PC that I have been
using for testing.  I have never checked to see how this GPS responds to the
DTR signal, I just assumed that it did.

-Josie
