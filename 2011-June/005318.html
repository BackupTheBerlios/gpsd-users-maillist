Yes.

> Then I can issue in the commands like the older version (a, etc) to get it to interrogate?

No.  The old commands are gone.  Instead, it will stream reports back at you 
like the old 'O' mode.  

The 'interrogation' commands in the old command set was badly broken - no
timestamps, for one thing, so you couldn't actually get any guarantee thar
multiple reponses pertained to one fix.

For producruib coder, you shouldn't actually be issuing your own commands
at all. Use one of the client libraries; it's simpler and safer.
