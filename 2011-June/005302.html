> > #if ( GPSD_API_MAJOR_VERSION >= 5 && GPSD_API_MINOR_VERSION < 0 )

> Version checking is done later in the code. GPSD_API_MAJOR_VERSION

That sounds reasonable. If the versions set in gps.h disagreed, it
wouldn't compile. What you are seeing is a run-time error. So perhaps
grabbing the latest source for gpsd and recompiling are in order?

Short of that, are there multiple instances of libgps.dylib on your
system? If so, perhaps the runtime code is picking up an older version.
