the pl2303 looks like a regular tty. baud rate change is application
dependent - ie. the standard termios interface applies.

skyout - you should install minicom. it's a nice curses terminal
program that lets you change port settings on the fly. also, minicom
doesn't select() so if there's anything on the device, minicom will
see it.

in gpsd, you could just read() on the device, even if select doesn't
say it's ready. add a few ms sleep in there to prevent spinning... i
did that on a friend's mac and it worked.

i know you said you saw the upstream bugs page, but i didn't see if
you tried the driver from prolific
http://prolific.com.tw/eng/downloads.asp?ID=31
