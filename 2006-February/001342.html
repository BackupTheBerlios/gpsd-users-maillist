Eric,
Well I've got the GPS::gpsd perl module up on CPAN.  It's currently at
version 0.3, but I'm already working on version 0.5. If there are any
perl fans out there I think you'll like it.  There's a "get" interface
and a subscription interface.  Properties are separated between server,
point and satellite objects like you would expect.

  use GPS::gpsd;
  $gps=new GPS::gpsd();
  $fix=$gps->get();

  $gps->register();

Now, I'll have to work on the documentation and installation stuff. 
The interface is currently not stable between version.

http://search.cpan.org/~mrdvt/perl-GPS-gpsd-0.3/

Ver 0.4 is up on
http://twiki.davisnetworks.com/bin/view/Main/GpsApplications
and pending CPAN.

I'd like to get v0.5 out soon because I like the subscription interface
much better.
