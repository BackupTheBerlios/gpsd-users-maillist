Now I talked with our hardware guru:

The GPS device comes with a gyro. The NMEA protocol was modified
to support those extended functions. The second mode field not only
include the three known values, but also the 'E':

1 = No fix
2 = 2D Mode
3 = 3D Mode
E = Estimated

'E' appears as soon as the device is not moving. And it don't
indicate an error or a "no fix" situation. I think the best
solution is to don't change the mode if an 'E' appears. The
patch attached at my last message should do that.
