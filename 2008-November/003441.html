I have just installed gpsd 2.37 on a CentOS machine (laptop).

Basic steps I used:
1, Download gpsd 2.37 from berlos.de website
2, Follow the INSTALL document, and "p=" command in "telnet localhost
2947" does return the latitude/longitude.
3, Run a daemon as root:
gpsd -N -n -D 2 /dev/ttyUSB0
