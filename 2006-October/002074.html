I reported about problems on disconnecting gpsd from my virtual serial
interface Moxa NPort 5210 (LAN-serial emulation). It freezes on
disconnecting.

I noticed now that it only happens if gpsd starts searching with the
wrong baudrate (for example 38400). If it starts with the right baudrate

starts with 9600 and sometimes with 38400.

Now the question is how to enable fixed port speed:
I tried configuring with

--enable-fixed-port-speed

which makes compile fail. I tried

--enable-fixed-port-speed=9600  (which is not documented correctly
in ./configure --help) and it compiles. But gpsd still starts with
38400.
