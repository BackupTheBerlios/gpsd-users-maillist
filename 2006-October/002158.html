On Tue, 31 Oct 2006, raphael Jacquot wrote:

> hi there
> I have a serial GPS16 here,
> have a couple questions :
> 1) how can I setup gpsd for garmin binary protocol instead of nmea ?

You need the latest SVN, then run this command before you start gpsd:

# garminctl -b /dev/XXXXXXX

garminctl is in a state of flux, so grab a copy now and save it in a safe
place.

> 2) is there a way to log the raw data coming in for later phase adjustments ?

Not quite yet.  the code to decode the phase data from the garmin is done.
There is no way to log it yet.  If you have a way to do the adjustments then
maybe you can suggest a data format?

> 3) do you know of any free software appropriate for that ?

Not me.  Chris Kuethe is workong on some for gpsd and knows of other
sources for the code.
