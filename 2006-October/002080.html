> I need some informations about USB connection of my GPS Garmin GPS-60.

You came to thr right place.

> At first: when I run gpsd I haven't any idea about socket in -F parameter.
> How I can find it? I have seen that many people use /dev/ttyUSB# but this
> solution don't run.

The -F is for hotplug. Skip that for now until you have the basics
working.  Unplug your GPS and then plug it in another USB port.  Then
run dmesg and see what the kernel says about the item.

If, for example, it said that the device was now connected as /dev/ttyUSB1
you would run this at the command line:

	killall gpsd
	gpsd -nN -D 3 /dev/ttyUSB1

Let us know how that woks for you.
