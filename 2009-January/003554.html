>> I have a GPS data logger i-Blue 747; is there any ways to get
>> gpsd working with this hardware (or: get iBlue 747 working with
>> gpsd) ?
>
> Yes, it is possible to use the i-Blue with gpsd via USB.
>
> Configure udev so it automatically attaches /dev/gps0 when you
> plug the device (may be Ubuntu already did it for you), and use
> that device with gpsd.
>
> Note that there are some caveats about gpsd and the USB device:
>
> - if GPS is not attached at boot time
> - if you unplug and replug the device during operations
>
> I wrote some notes and suggested two workarounds here:
> http://www.rigacci.org/wiki/doku.php/doc/appunti/hardware/eeepc_navit#installing_the_gps_daemon
>
> But I think that we should file a bug against gpsd.
