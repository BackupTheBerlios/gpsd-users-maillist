I just want to inform that Holux M-1000C works with gpsd 2.37. There
is no page on the manufacturer web site for this device (yet), but there is
some information[0] for M-1000 and M-1000B which I think will works
with gpsd too as stated in this[1] page.

These devices are Bluetooth SPP compatible. The M-1000C however
using MediaTek MT3329 chip and emits NMEA 3.01. I'm not sure
about baudrate-hunting[2] bug though.

[1] http://www.linux-gps.org/index.php5?title=Holux_M-1000
[2] http://gpsd.berlios.de/upstream-bugs.html#bluetooth
