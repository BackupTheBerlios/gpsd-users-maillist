On Thursday 08 December 2005 01:51, Andrew Frazer wrote:
> Hi, I am looking to find some software that will allow me to track a number
> of gpsd sources simulatenouly, and output the result to a webpage, (over a
> map perhaps)  It would be good to be able to store the data of each device
> so the track could be stored.
>
>
>
> Any pointers would be greatly appreciated.

If you don't need the results on a webpage you can use gpsdrive with its 
friendsd. The friendsd is a server which collects all positions from clients 
and sends them back out to all connected clients. All positions are then 
displayed on the map-display of gpsdrive. Friendsd uses UDP for 
comminication.
Current Versions can be found at 
http://www.ostertag.name/gpsdrive
