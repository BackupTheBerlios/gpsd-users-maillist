Matthew Perry wrote:
> Hey folks,
>
>   I am looking for a python module to talk with gpsd. I installed gpsd
> on debian and noticed that it came with a gpsd.py module but I can not
> seem to get it working.

I am writing an application for my car using wxPython. I can tell you
that gpsd.py works :)

> xgps and gpsdrive also work fine. But when I try to connect using the
> python module, I'm not getting any results:

>>>>import gps
>>>>session = gps.gps()
>>>>session

> This is probably an oversight on my part but I have not been able to
> get past this step despite a good amount of time experimenting and
> googling for answers.

You need to send a query to gpsd so it will save the data. In my
application I send a gps.query('mos') every 10 seconds or so. Try:

  import gps
  session = gps.gps()
  session.query('mos')
  session
