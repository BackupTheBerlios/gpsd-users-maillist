Hi,

I'm trying to build a code using both 2.39 and 2.90 gpsd library. In
order to activate correct part of code, I wrote a macro:
#define GPSD_CHECK_VERSION(major,minor)    \
   (GPSD_API_MAJOR_VERSION > (major) || \
    (GPSD_API_MAJOR_VERSION == (major) && GPSD_API_MINOR_VERSION >= (minor)))

Then I just realized that:
- major and minor does not follow the version of the project
(disturbing, but why not)
- nor major nor minor was updated when gpsd goes from 2.39 to 2.90.
It is quite annoying because I think a notable change occured in API:
gps_open_r. Furthermore, gps_open now return a static structure, so a
single call is legal, while 2.39 returns a newly allocated struct.

Hope that helps.
