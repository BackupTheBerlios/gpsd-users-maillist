It is possible to use at least some client software for Windows to connect to 
remote gpsd. You have to install virtual serial port software that is used to 
tunnel serial communication inside TCP/IP. Than you have to use miniterm to 
send "R" and close it without disconnection. After that it should be possible 
to hook other software (VisualGPS in my case) to the NMEA data stream.
