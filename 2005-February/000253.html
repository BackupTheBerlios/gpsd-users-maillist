Mike T. Machenry <dskippy at ccs.neu.edu>:
> I have an Apple iBook G4 12" and I've recently obtained an Earthmate USB GPS 
> device. I hear that support for this device is not so good but unfortunately 
> beggers can't be choosers and I got it for free. Apparently gpsd supports 
> this device but I have a few questions about it.
> 
> 1) The hardware part of the website says it requires Linux 2.6.10 at least 
>    for Cypress support. Does this mean I need 2.6.10 for Cypress but I can 
>    the device working without Cypress or is Cypress support neccessary for   
>    the device to work at all?

Cypress support is necessary for it to work at all.  I think I remember 
hearing that OS X has working Cypress drivers.

> 2) I am trying to make this device work in Mac OS X (10.3). gpsd builds fine 
>    and runs, but I need to pass it a device file as an argument. I do not 
> get a device file in /dev when I plug the in the earthmate. Is there a way I 
> can create a device or do I need a driver that's not availible for Mac OS to 
> do so. How much of the communication with the earth mate is done in gpsd in 
> other words? Can I just pass a raw device file to gpsd to get it to work?

Yes, just pass it the raw serial device file.
