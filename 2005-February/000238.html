Hi Arne!

> Are there any tools to switch from SiRF to NMEA under Linux?

Some time ago I wrote a tool for the Holux GM-270 GPS CF card. Among
other things it allows switching from NMEA to SiRF binary mode and vice
versa.

   http://home.knuut.de/arnim.laeuger/sirf_ctrl-0.1.tar.gz

It is quick & dirty, poorly documented and only tested on my Zaurus PDA
so far. However, being a command line tool there should be no major
issues "porting" it to a full blown linux box.


Good luck
