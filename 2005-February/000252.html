Look at the source code for FlightGear (www.flightgear.org).  FlightGear
is a flight simulator that has a pretty impressive I/O mechanism.  You
can tell FlightGear to output data in several different formats
(including NMEA) on a device or network port that you choose.  Unless
I've managed to misread the code, you can tell FlightGear to output NMEA
data on a serial device.  This is probably way overkill for what you
need, but it could give you an idea of how it can be done.
