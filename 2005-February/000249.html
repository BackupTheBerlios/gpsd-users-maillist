Hello!

Some time ago I found a small program named "Cutecom". This is a serial 
terminal aimed at developing serial devices as microcontrollers with  a 
serial port. The program can be used as replacement of minicom (which is 
rather aimed for modem operation). 

Recent releases of Cutecom (since 0.13) have hex input and output mode so they 
can be used to investigate binary format of some GPS devices.

The Cutecom is hosted at Sourceforge (http://cutecom.sourceforge.net) and uses 
Qt library.
