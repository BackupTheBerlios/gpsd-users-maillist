I wrote a C program to log data from gpsd to use for my senior
project. I'm trying to make a tool to roughly gauge and track
car performance using gps data. There already are a few
commercial devices that do this very accurately but they are
pretty pricey. My goal is to make a low cost (price of gps
mouse) solution for hobbyists, ie me.

This logger only records time, position, and velocity but it
could easily be modified to gather other information. The code
could be prettier, formatting and hard coded values in
particular, but it's much better than my first version that I
accidentally deleted. If I have any glaring problems let me
know.

usage: ./gpsl [duration of log in seconds]

http://www.csc.calpoly.edu/~dhphan/gpsl.tar
