The gps 18 usb version does not transmit NMEA, but garmin's proprietary binary 
protocol.  To use this with gpsd you should check out the below link blow.  
It contains the linux kernel driver and the garminusb2nmea application which 
grabs the data from the kernel driver and translates this into an NMEA stream 
that gpsd can understand.  Of course this will require a fairly recent kernel 
since the driver is being developed against 2.6.10, or it might even be 
included with the recent kernel.  The instructions that come with the 
packages should explain how to install them.

http://sourceforge.net/projects/garmin-gps/
