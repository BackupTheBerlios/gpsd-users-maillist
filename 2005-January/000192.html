A less spartan version of this approach is to use minicom or gtkterm to
log the NMEA stream to a file.

Perhaps this task is good gpsd+perl/whatever minimalistic reference
implementation for the masses of amateur tinkerers out there.


Many Garmins can log "x y time" to a track file, eg every 30 seconds.
At the end of the day you can download the track to a text file with
something like gpstrans or gpsbabel.
