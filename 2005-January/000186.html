On Tue, 11 Jan 2005, Ragnar Wisløff wrote:

> The Garmin eTrex Legend C specs only say it will output "Garmin data" on its
> USB connector, nothing about NMEA. Has anyone succeeded (or failed) in making
> it work with gpsd?

I recently got my new Garmin GPS-11 to work with gpsd.  It also sends
"Garmin Data" over USB>

You need the Garmin USB (garmin_gps) kernel driver and Garmin to NMEA daemon
(garminusb2nmea) rom here:
	http://sourceforge.net/projects/garmin-gps/

I have been using garmin_gps version 0.19 on Linux 2.6.8.1

There is a newer 0.22a that has some recent fixes but those require
Linux 2.6.10.  Not been able to get anything past 2.6.8.1 to run on my
laptop yet.  Even 2.6.8.1 has been downhill from 2.6.6 for me.
