Following up on earlier report....

The current version of 'xgps' had a bug whereby the display of 'Speed' when 
the selected units are 'knots' was incorrect. This bug appears to have been 
introduced back in Apr 2005 when the 'O' command of 'gpsd' was changed to 
report speed in meters/sec.

This bug is now fixed in the latest SVN (r3861).
