IMHO this doesn't belong in GPSD.

TMC data is not something received by the GPS. RDS-TMC data is, as
Chris right stated, metadata regarding the current traffic situation.
It's an european system, where traffic situation metadata is encoded
and transmitted wih standard FM radio station signal. The metadata can
then be used in conjunction with a compatible navigation program, so
for instance when there are highway jams, traffic accidents, etc, the
navigation program can automatically reroute the traffic.

You need a seperate TMC receiver on the FM band, to get the RDS-TMC
data stream, and it is not a NMEA or similar compatible protocol.
Also, AFAIR you need to pay a licensing fee, in order to get the
decoder specs - so not an open protocol.

More info : http://www.rds.org.uk/episode/2_4.htm
