I think we'd like to give everyone a Christmas/Northern Hemisphere
Winter Solstice present and ship gpsd 2.34 before the start of the new
year. The tree is now locked. This means:

- No new features in core code. [1]
- Minimally intrusive bug fixes only. [2]
- No refactoring or architectural changes.
- Documentation patches are acceptable.

[1] An exception to this might be made if someone wanted to contribute
a GTK or QT gui client. I won't guarantee it'll ship, but I will
consider it.

[2] If you're unclear about what constitutes a mimimally intrusive bug
fix just send it our way. We're all about bug fixing, and if that
means we need to touch every file in the tree, so be it. But we would
like the tree to stabilize and get all the bugs shaken out.

Gary is alive, albeit very busy right now. If anyone happens to fix
PPS before Gary gets less busy, please send diffs to Eric or myself.

Testing and bug reports are encouraged. To put it another way: if
you're following current, what's broken for you? We don't mind hearing
from 12 different people that XYZ is broken, that gives us some idea
of the scope of the problem.
