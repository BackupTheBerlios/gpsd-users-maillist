I was trying to use libgpsmm to connect to several GPSs simultaneously.

that gpsmm::open() uses gps_open() rather than gps_open_r(), which would

keyword for the gpsdata struct).

Why is that?

For the time being, I switched to using the C interface with 
gps_open_r(), and it works very well.
