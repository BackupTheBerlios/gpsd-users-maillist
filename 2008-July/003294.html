1. The GPS model name.

Blumax GPS-009

2. Whether or not it works.

Almost out of the box  - see notes for details.

3. If possible, an URL to a technical reference on it.

http://www.eurotronic.net/products/produktdetails/gps_receiver.html

4. The GPS chip it uses.

SiRF Star III (according to data sheet)

5. The interface type (RS232, USB, CF, UART (= 5V TTL RS-232))

Bluetooth

6. The vendor

Eurotronic Products GmbH  (via amazon.de)

7. Version of gpsd you tested with

2.36

8. NMEA version it emits, or '?' if the documentation doesn't say

V3.01

9. Notes on the device

Requires "-b" ... i had to totally drain the devices battery
and let it rest for a few days before i was able to use it again
after a first attempt of using it with without "-b"

$ gpsprof -f cycle

Cycle report Fri Jul 18 16:22:21 2008, Generic NMEA, 9600N1, cycle 1s
The sentence set emitted by this GPS is: GSA RMC GGA GLL GSV
GSA: is probably emitted once a second.
RMC: is probably emitted once a second.
GGA: is probably emitted once a second.
GLL: is probably emitted once a second.
GSV: is probably emitted once a second.
Send cycle is once per second.
