Cowart, Donald wrote:
> I understand how to query & connect to a local gps in python with
> something like;
> 
> import gps
> sess = gps.gps()
> 
> and then making queries with the sess object.  How do I make a
> connection to a gps running on a remote host though?

Easy, just pass the host parameter:

sess = gps.gps(host="your.remote.host")

This even works with a DNS name or an IP address (which you also write
as a string)
