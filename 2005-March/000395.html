What is EMUL?

 The EarthMate Userland Library provides access to the usb earthmate gps 
device via libusb and libpthread.  EMUL aims to be as portable as possible in 
order to function with the various operating systems libusb supports.

 What software is supported so far?

 The project is releasing patches for recent versions of the gps daemon. 
Release 1.0.1 contains a patch for gpsd-2.17 located in the source archive 
under directory 'patches' along with instructions on how to apply.  Please 
refer to the README files.

 Which operating systems are supported so far?

 So far EMUL has been tested under gentoo linux, a debian based distribution.  
Other target distributions for testing include Fedora Core 3, Mandrake, etc.  
FreeBSD, OpenBSD, and open/gnu darwin (since I do not have access to Mac OS 
X) are as of yet untested.  Reports of success would be much appreciated.
