Hello all,

First, thanks a lot for the effort of making gpsd, very usefull project, 
clean design...


Here is my report of using gpsd with the Techway TH-011 GPS device:

1. The GPS model name

Techway TH-011


2. Whether or not it works

Not perfectly

gpsd manage to open the device and read MNEA sentences
but never get any fix ( although the demo program get fixes in the same 
conditions)

Here is the log from the demo program:
http://jeanchristophe.duber.free.fr/GPS/G_Port2.log



3. If possible, an URL to a technical reference on it

http://www.techwayinc.com.tw/support.htm#2


4. The GPS and (if applicable) USB-to-serial chip it uses. (Look at the 
output of lsusb(1), it may identify the USB-to-serial chip. If it gives 
a vendor and product ID, send us those numbers.)

5. The interface type (RS232, USB, CF)

USB ( with an external PS2(RS232) to USB converter )


6. The vendor

Techway, http://www.techwayinc.com.tw/


7. Version of gpsd you tested with

2.17


8. NMEA version it emits, or '?' if the documentation doesn't say

?


9. A list of NMEA sentences it emits


$GPGSA
$GPGSV
$GPRMC
$GPGGA

Notes:
* The specification says: GGA, GLL, GSV, GSA, RMC, VTG
* It emits the two sentances at the begining: $PFEC and $PSRF105


10. Notes on the device

cheap GPS device ( between 50 and 60 € )
claimed to be Waterproof
Base Band TECHWAY SOC, AB0201
