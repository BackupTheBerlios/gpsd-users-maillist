As promised, I made fixing disconnect/reconnect my top priority.  This
turned out to require tearing apart the packet engine and rebuilding
it, since the problem was triggered by disconnect during the blocking
reads it had been using to fill the packet buffer.  

SVN head has the new code in it.  Please test and report any problems.

Gary has reported that the Garmin driver is busted.  It's probably
some minor integration problem triggered by recent changes.  Once that
is fixed, we can ship 2.17.
