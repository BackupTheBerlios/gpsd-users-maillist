On Fri, 16 Mar 2007, David Cottle wrote:

> Many thanks!  Indeed I tried ntpdc -p and I don't see my gps.
>
> Its definately in my ntp.conf file as I pasted below.

Here is mine, looks like yours:

# for gpsd
server 127.127.28.0 minpoll 4 maxpoll 4
fudge 127.127.28.0 time1 0.429  refid GPS
# for PPS and gpsd
server 127.127.28.1 minpoll 4 maxpoll 4 prefer
fudge 127.127.28.1 time1 -0.0004 refid GPS1

Some distros play games with the ntp config file name.  Recheck that
ntp.conf is the right one for you.  Then it's time to compile from
scratch.

> As I am running Fedora (FC6) any tips on compiling a new ntpd for my
> server?

I prefer Linux myself and stay away from Fedora.

Here is how I build my ntp:

	./configure --disable-nls --with-crypto
	make && make install
