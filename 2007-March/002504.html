I'm thinking it's time to cut a new release; some {bugs, misfeatures,
significant sources of annoyance} have been addressed in SVN, and it's
probably time to ship.

Things to note:
* more robust nmea parsing
* improvements in the geoid handling calculation
* 3 new drivers
* the OS X problem was traced back to old drivers
* there is now a runtime flag for bluetooth safety (aka. read-only devices)
* prototype driver is closer to actually compiling
* more documentation

So... now's the time to test. Let us know if things don't compile or
crash. As far as I know, we've got most of the kinks worked out, but
it'd be really great if we didn't have to slap in a bunch of patches
right after release.
