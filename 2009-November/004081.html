I managed to fix the problem by removing and reinstalling autoconf and
automake.

I successfully built gpsd using autoconf-2.59 and automake-1.9.  These
are earlier versions than what I was using before, and they're earlier
than the versions recommended in the INSTALL file, but all I can say is
that autogen.sh worked (for me) with these versions and did not work
(for me) with the later versions I had tried earlier.
