BTW: while trying to figure out what my old Trimble SVeeSix was doing, I
wrote a minimalist perl TSIP packet dumper.  It is not complete:

- it mostly only decodes packets I've seen from my SVeeSix
- it doesn't try to parse any data except for converting GPS time to UTC
- error checking is pretty minimal (if you start it in the middle of a
  packet you may need to kill it and try again)

However, it could be a useful debugging tool for anyone looking at TSIP.

http://www.cmadams.net/misc/gps/tsip.pl-1.0

It only reads packets; if you want to send commands, you'll need to open
another window and echo packets out manually.

It is based on this Trimble TSIP reference manual:

http://trl.trimble.com/dscgi/ds.py/Get/File-6222/TSIP%20Reference%20Manual%20-%20Rev%20C.pdf
