Chris <chris at maildrop.hobby-site.com>:
> Autobaud means it uses the speed the port is set to, or uses the speed the 
> connected device uses? So applications using (or like GpsDrive even starting 
> gpsd by themself) have to set the serial port speed by themself.

No.  What gpsd does is try the existing port speed first. If that
doesn't yield recognizable GPS packets, it cycles through 4800, 9600
and higher speeds.  At each speed it watches for a well-formed packet
an any of the types it supportes.
