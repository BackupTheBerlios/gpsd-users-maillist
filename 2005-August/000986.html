As was discovered in the recent gpsd/Zaurus thread, there is a problem
with Semson's i.Trek M3 Sirf-III-based GPS receiver: the NMEA GPGSA line
is missing fields when there is no fix; instead of returning blank
satellite fields, it simply excludes them.

Semson's has informed me that this is a problem with *all* Sirf-III
receivers (assumedly due to the fact that they all use the same base
firmware). Fortunately, they provided me with an updated firmware (v.
3.1.1) which fixes the problem.
