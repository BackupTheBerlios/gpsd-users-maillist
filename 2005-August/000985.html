Have it running in 60 seconds:

Put this in /etc/ntp.conf:

server  204.123.2.5             # clepsydra.dec.com
server  204.152.184.72          # clock.isc.org
server  17.254.0.49             # time.apple.org
server  207.46.130.100          # time.windows.com
enable auth monitor             # enable the good stuff
driftfile /etc/ntp.drift        # path for drift file
authenticate no

Then run this:

ntpd -g -l /var/log/ntpd.log

You can see it working this way:

watch ntpdc -p

That's all there is to it!
