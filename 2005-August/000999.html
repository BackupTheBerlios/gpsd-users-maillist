I've got a copy, but it dates to 1 Jan 2002, so it's likely not the
latest and greatest.  While I won't quote the document verbatim out of
fear of legal action from the Maritime Association, feel free to ask
specific questions and I'll try to describe what the spec says.

For example, the spec says the last field is called the 'Mode Indicator'
field, and can assume the following values:  A is Autonomous mode,  D is
Differential mode,  E is Estimated (DR) mode,  M is Manual Input mode, S
is Simulator mode and N means "Data not valid".  And the spec says that
"All data fields must be provided, null fields used only when data is
temporarily unavailable".
