Another problem I found is with the construct:

"mmsi":211315540

This defines the "mmsi" as a number. The problem with this is that I
found mmsi like this: 002442024. Some languages (python for instance)
read this as an octal number.

Maybe it is wise to make the mmsi a string.

"mmsi":"002442024"
