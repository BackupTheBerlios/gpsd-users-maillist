> I only know Alt-Shift-SysRq- [e,i,u,s,r,b] for forcing a reboot.

You can see all of the supported sysrq combinations by looking in the
linux/drivers/char/sysrq.c file. Search for sysrq_key_table.

If you do SysRq and an invalid letter (like X) then it will print out
the available options.

On my ARM processor, T shows all of the running tasks and the
registers for each task (including the PC). SysRq-X gives me this
output:

Dec 31 16:00:29 kernel: SysRq : HELP : loglevel0-8 reBoot tErm kIll
saK showMem Nice showPc unRaw Sync showTasks Unmount

My kernel isn't quite the most recent and doesn't have the SysRq-D
showlocks thing, so I'm not quite sure what the output looks like.
