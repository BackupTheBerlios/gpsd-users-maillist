I just wanted to give everybody an update on what I've got so far.
I have a Garmin Forerunner 305 connected via USB to my Ubuntu 6.06 laptop
and am running the stock gpsd which is version 3.30
It is using the garmin_gps driver and is apparently speeking the garmin
protocol. If there is a way to change the protocol on this device I can't
find it.
gpsd seems to be communicating with the device just fine. If I run in debug
mode I can see NMEA sentences, presumably translated from the garmin
protocol, and the only data type that I am seeing though is gpgsv. I think
that I should be seeing other data types that actually give location
information, but I'm not sure and certainly don't know how to get it to send
them. The device does have a fix while I'm watching this, but even when it
doesn't have a fix I still see these same packets.
If I run gpsdrive without first starting gpsd then it detects a garmin mode
device and give me accurate location information without gpsd at all so I
know that this information is available from the device.
I know that this is not a supported device, but I'm not sure why it
shouldn't work. I'm not an expert coder, but I am willing to put some work
into this if it looks like it'll help.

Thanks in advance for any help,
Mike Robbert
