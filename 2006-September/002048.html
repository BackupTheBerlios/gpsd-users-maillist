What debug level did you run this at?

At debug level 5 or above you should see a messsage like this

gpsd: pps-detect (DCD) on /dev/ttyS1 changed to 1
gpsd: ntpshm_pps: precision -11

If you see the "precision" then PPS has been sent from gpsd to ntpd.
