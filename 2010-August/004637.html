Yo Volker!

On Thu, 19 Aug 2010, Volker Wambach wrote:

> On OS 11.2 i had no problems with gpsd an my GPS-logger Wintec G-Rays 2.
> gpsctl says: ... identified as Generic NMEA at 57600

Is it NMEA at 57600?

If so, you should be able to do this without gpsd running and see NMEA:

	cat /dev/ttyUSB0

If that shows you NMEA, then try to run gpsd with a higher debug level
like this:

	gpsd -nND 6 /dev/ttyUSB0
