A lot of good stuff (and some significant bug fixes) have landed in
the last month, and I'd like to ship a 2.93 point release in a week 
or so.   We're in state 1 (blue) and I'm awaiting confirmation that
we have good operation on the problem children (Trimble and Garmin).

The bugtracker on Berlios looks pretty clean.  Known issues we
should resolve before release if possible:

* Jiří Techet <techet at gmail.com> has reported gpsd eating 100% of
  CPU under some circumstances, and given a recipe for reproducing
  the problem. I think this may be identical to Berlios tracker
  bug #14638: 100% cpu when Bluetooth gps device vanishes, which I
  couldn't resolve because it wasn't clear how to reproduce it.  It's
  going to be my top priority before release.

* gpsctl -b should work on UBX, but does not. This one is ckuethe's.

* The PHP test client needs work to play nice with the new JSON-based
  test protocol.  This one is ckuethe's too.

If you have any bugs or issues that you think are urgent, speak up now.
