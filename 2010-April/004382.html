My gps receiver shows up as the Prolific Technology, Inc. PL2303
because that's the RS-232 to USB adapter chip the manufacturer used.

> 4. run sudo modprobe usbserial vendor=0×XXXX product=0×XXXX
> 5. run gpsd dev/ttyUSB0

Are you sure there is a device there?

Also, the command would more likely be:

sudo gpsd /dev/ttyUSB0

Note 1) the leading slash in the device path, and 2) you must run gpsd
as root.

However, on Ubuntu, the following command takes advantage of some of
the operating system goodies:

/etc/init.d/gpsd start
