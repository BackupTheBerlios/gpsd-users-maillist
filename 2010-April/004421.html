Hi all,

For a few years now I've been synchronizing my ntpd to a Garmin GPS 18  
LVC and running gpsd to create a skyview web page. I started doing  
this on a Debian etch system with gpsd v2.36, which required a patch  
(by John Arthur, Sat Jan 12 06:09:54 CET 2008) to work.

For gpsd to work after the upgrade to Debian lenny, I had to keep  
running v2.36, because there is no similar patch that I know of for  
the subsequent versions. Unfortunately, this apparently fragile  
configuration finally broke after a Debian update I had been  
postponing for a while: ntpd is still synchronized to the Garmin, but  
gpsd can no longer obtain any information for the skyview. I tried a  
backport version -- v2.39 -- but it made no difference.

Any suggestions towards a solution would be appreciated.

Cheers,

Jaap
