Hello, are BINR, RTCM SC 104 data protocols supported by gpsd ?

The BINR, RTCM SC 104 protocols is transmitted by the CF-4706 chip used 
in the CH-4711 GLONASS/GPS mouse:
http://www.nvstech.ch/?page_id=230

By default the device does not report 2d fixes; the vendor configuration 
tool offers checkboxes to enable any combination of none, 2D fixes, 3D 
fixes, or both. The devices has only a very limited set of NMEA controls 
but speaks a proprietary vendor format called BINR with more capabilities.
It is supported by gpsd ( NMEA data)

nvstech, that is the european vendor for Navis can be reached at:
info at nvstech.ch

the maker of the chip can be reached at :
navis at navis.ru
