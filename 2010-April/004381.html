> hello. I'm new to gpsd and ubuntu.
> 
> I just want to run gpsd on my system. I have a GPS antenna, the circuit
> board needed for it, a 9V battery and a serial-to-usb convertor cable.
> 
> My steps were:
> 
> 1. Install gpsd trough synaptic
> 2. plug gps-antenna to usb-laptop
> 3. run lsusb command, to find out the vendor and product ID
> 4. run sudo modprobe usbserial vendor=0×XXXX product=0×XXXX
> 5. run gpsd dev/ttyUSB0
> 
> ...still no output.
> 
> can somebody tell what I am missing ?

Well, if you run gpsd /dev/ttyUSB it's going to go to background
and you won't see any output immediately. Look in the TROUBLESHOOTING
file in the package doc directory for sep by step instructionds 
- it's called INSTALL in the source distro, but they changed the name.
