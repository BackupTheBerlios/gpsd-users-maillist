> If this in binary format how to decode?

"normal = 9600, GGA+GSA+GSV+RMC+ZDA\n"

That's NMEA at 9600bps. If you connected at 4800 that would produce
strange characters. You're using minicom so it's easy to check all the
other common speeds.

Oh - one thing I've noticed. Sometimes these usb-serial cables get
confused if you're sending data while plugging them in. This too can
lead to strange characters. The workaround is to plug in the
usb-serial, then power-on or connect the device that is sending data.
