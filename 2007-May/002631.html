On 5/9/07, Randy <wrr01 at yahoo.com> wrote:

> Is there a utility that will display the unlinked archives and can link everything?  or, is libtool
> not being passed the correct parms to cause the complete static link to occur?
>
> if i can get this lib to link i should be there or at least very close!

Libtool is getting it wrong. Even if you set your CFLAGS and LDFLAGS
to have "-static", libtool just eats those. I haven't yet found a way
to make it not do that... ok, there is the "-all-static" flag to
libtool but there doesn't seem to be a libtool_flags environment
variable you can pass in (already tried).

A slightly ugly solution is to pay attention to the gcc command that
is run at the link stage. Just run that by hand with the addition of
-static.
