On 5/7/07, Ran Talbott <ran at netgate.net> wrote:

> If I kill gpsd and do a cat /dev/ttyUSB0,  I can see that NMEA data is
> still coming,  but gpsd appears to have simply lost interest in it.

You are running gpsd without "-n"; this means that gpsd will open the
device, identify it and then close the device if there are no clients
connected. This is a power saving feature. If you would prefer that
gpsd opens the device and camps on it, run with "-n". This behaviour
is documented in the manpage.
