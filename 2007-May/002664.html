David Anullo wrote:
> i use gpsd (with linux 2.6.18) and i want to redirect the log that i 
> obtain with "-D4" option on an external file (like "cat /dev/ttyS0 > 
> file.txt"). I tried with ./gpsd -N -D4 /dev/ttyS3 > log.txt but the
> log file is empty..

if log messages are sent to stderr, try:
  ./gpsd -N -D4 /dev/ttyS3 2> log.txt


"... > log.txt" redirects only stdout.

for more info search around for `tee`, "1>&2", pipe, shell redirect.
